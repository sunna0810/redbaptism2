# 붉은 세례: 침묵의 방주 (Red Baptism: Ark of Silence)

침입형 SF 사이버 미스터리 텍스트 방탈출 게임. 실종된 사회부 기자 박지수의 단서를 찾아 종교 집단 '광휘의 방주' 본거지로 침투하고, 냉하한 스마트 글라스 인터페이스를 통해 비밀 단서들을 찾아내세요.

---

## 🌌 스토리 개요

> **"두려움은 자유의 착각입니다. 방주 안에서는… 모두가 평온합니다."**

사회부 기자로서 전쟁터와 사고 현장을 누벼 온 당신에게 한 통의 무전이 도착합니다. 실종된 동료 박지수가 위장 잠입해 있던 사이비 교단 **'광휘의 방주'** 예배당. 그곳에서 흘러나오는 붉은 조명과 성가 소리 사이로 이상한 냉기가 흐릅니다. 

당신은 안전망이 없는 차가운 지하 침전지로 걸어 들어가며 스마트 글라스의 전원을 킵니다. 
진실을 지상으로 송출해 동료를 구하고 이 거대한 사기극을 폭로할 것인가, 혹은 이 침묵의 방주 속 지층 아래에 영원히 묻힐 것인가.

---

## 🛠 아키텍처 및 폴더 구조

본 애플리케이션은 원본 단일 HTML 게임 로직을 현대적인 **React 18+**, **TypeScript**, **Tailwind CSS** 기반의 복합 모듈 아키텍처로 컴포지션 분리하여 이식한 완벽한 템플릿입니다.

```bash
├── index.html              # 최소화된 진입 마크업 (디스플레이 폰트 사전로드 처리)
├── package.json            # Vite 및 모던 패키지 리스트
├── metadata.json           # 앱 스펙 및 전역 메타 정보 설정
├── README.md               # 본 설명서
└── src
    ├── App.tsx             # 중앙 Game State 조율자 및 인터랙티브 라우팅 트리거
    ├── main.tsx            # React 마운트 지점
    ├── types.ts            # 게임 내 전역 데이터 모델 정의 (Room, Puzzle, Dialogue 등)
    ├── index.css           # Tailwind + 커스텀 테마 연동 허브
    ├── data
    │   ├── gameData.ts     # 예배당(B1)부터 심장부(B4)까지의 구역 데이터셋, 퍼즐 정보, 대화 맵
    │   └── endings.ts      # 최종 진행도 및 가치관에 따른 멀티 엔딩 에필로그 정보
    ├── engine
    │   ├── state.ts        # 상태 초기화 및 불변 복사본 복제 제어 장치
    │   ├── gemini.ts       # 구글 Gemini-3.5-Flash 모델 연동 대화 보완 장치
    │   ├── dialogue.ts     # 분기형 대선택지 라우터 및 전제조건 분석 장치
    │   ├── events.ts       # 추격전 타이머, 문전 대치 이벤트 및 상태 트래커
    │   ├── puzzles.ts      # 키패드 자물쇠, 무선 다이얼 조율 등 퍼즐 검증 알고리즘
    │   └── ending.ts       # 신뢰성, 호전성 수치 분석 기반 자동 엔딩 분기 연산기
    ├── renderer
    │   ├── hud.tsx         # 배터리 바, 카운터, 인벤토리 HUD 서브패널
    │   ├── rooms.tsx       # 룸 실루엣 아트워크(SVG) 및 핫스팟 맵 레이어
    │   └── screens.tsx     # 타이틀, 인트로, 대화 다이얼로그, 퍼즐 팝업, 엔딩 크레딧 슬라이더
    └── styles
        ├── main.css        # 분위기 오버레이(grain/scanline), 레이아웃 전반 스타일
        └── zones.css       # 구역별 분위기 셰이더 및 플리커 애니메이션 CSS
```

## 🧩 주요 퍼즐 가이드

1. **MAIN QUEST 1  ·  예배당의 암호 (금고)**
   - **단서**: 성가 악보 3페이지 (도·미·솔·시 음표 밑줄), 기도문 액자 뒷면 (*"빛은 침묵 속에서만 말한다."*)
   - **해법**: 음계를 매칭하여 `1357`을 입력하고, 레버를 밀어 넣기 전 하단 콘솔의 **'침묵 모드'**를 강제로 켜야 발각되지 않고 열립니다.

2. **MAIN QUEST 2  ·  생활관의 유품 (47번 침대)**
   - **단서**: 벽면 침대 배정표 일치 대조 (노윤아는 46번이라 우겼으나 실상 위장명 이준혁은 47번 배정)
   - **해법**: 침대 그리드에서 정확하게 `47`번 침대를 터치하여 매트리스 아래 봉투를 확보하세요.

3. **MAIN QUEST 3  ·  집행 구역의 진실 (무전 채널)**
   - **단서**: 환기구 안 소형 수신기 획득
   - **해법**: UP/DOWN 버튼을 통해 다이얼을 `CH 7`으로 맞춰 도청에 성공하십시오.

4. **MAIN QUEST 4  ·  심장부의 서버 (터미널 패스워드)**
   - **단서**: 흩어진 메모 조각 4장 수집 (책상: `ARCH` / 화장실: `E0` / 출입문: `1` / 책장: `10`)
   - **해법**: 메모를 구역 순서대로 병합하여 `ARCHE0110` 패스워드를 입력하면, 백업 시퀀스와 동시에 박진감 넘치는 20분 자폭 프로세스가 시동됩니다.

5. **MAIN QUEST 5  ·  탈출과 송출 (나침반 방향 설정)**
   - **단서**: 오리엔테이션 미궁 비상구 착오 간섭 배제
   - **해법**: 스마트 글라스의 나침반 바늘의 진북 지점인 `N`방향 비상구를 빠르게 관통해 탈출에 돌입하세요.

## 🧠 AI 동적 인터랙션 (Gemini Link)

타이틀 하단의 **"Gemini API 키 입력"** 버튼을 눌러 발급받은 Google AI Studio API 키를 등록하면:
- **노윤아, 교주 박규빈, 편집장 김재원** 등의 무전 및 대면 발언이 현재의 구역, 누적 신뢰 및 의심 상태에 따라 실시간으로 구문을 가변 변경하여 더욱 웅장한 공상과학 체험을 선사합니다.
- (API 키가 없어도 정적으로 완전하게 정주행 패키지 플레이가 가능합니다.)

## 🚀 로컬 실행 방법

애플리케이션을 빌드하고 실행하려면 터미널에 아래 명령을 입력하세요:

```bash
# 1. 패키지 종속성 설치
npm install

# 2. 로컬 개발 서버 기동 (Vite)
npm run dev
```

브라우저에서 `http://localhost:3000`으로 접속하여 차가운 붉은 침묵 속으로 침투해 보세요.
